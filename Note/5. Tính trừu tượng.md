Tính trừu tượng trong OOP
---
- Trừu tượng (Abstraction) là nguyên lý OOP cho phép che giấu chi tiết cài đặt bên trong và chỉ bộc lộ những đặc điểm, hành vi quan trọng cho người sử dụng.
- Chỉ cho người dùng biết cái gì cần làm mà không cần biết làm thế nào
- Công cụ: abstract class và abstract method
- Dễ bảo trì, thay đổi bên trong mà không ảnh hưởng code bên ngoài

```
	# Lớp trừu tượng
	class Animal(ABC):
	    @abstractmethod
	    def make_sound(self):
	        pass
	
	# Lớp con cụ thể
	class Dog(Animal):
	    def make_sound(self):
	        print("Gâu gâu")
	
	class Cat(Animal):
	    def make_sound(self):
	        print("Meo meo")
	
	# Sử dụng
	animals = [Dog(), Cat()]
	
	for a in animals:
    a.make_sound()

```
-> Animal là abstract class -> chỉ định rằng mọi con vật phải có phương thức make_sound()

Lớp trừu tượng
---
- Không thể tạo đối tượng trực tiếp từ abstract class (lớp trừu tượng)
- Lớp trừu tượng có thể chứa:
    - Phương thức trừu tượng -> bắt buộc lớp con override
    - Phương thức bình thường -> lớp con trực tiếp dùng lại hoặc override
 
```
from abc import ABC, abstractmethod

class Shape(ABC):
    @abstractmethod
    def area(self):
        pass

    def greet(self):
        print("This is a shape.")

class Circle(Shape):
    def __init__(self, radius):
        self.radius = radius

    def area(self):
        return 3.14 * self.radius ** 2

c = Circle(5)
print(c.area())   # 78.5
c.greet()         # This is a shape.
```
- Nếu lớp con không viết lại tất cả abstract thì nó cũng sẽ thành abstract class -> không thể tạo đối tượng
- Nếu lớp con có những thuộc tính riêng và vẫn muốn dùng lại cả thuộc tính của lớp cha, thì phải có phương thức khởi tạo (constructor) riêng, trong đó gọi constructor của lớp cha.
- Lớp con gọi constructor lớp cha bằng super().__init__(x)
```
class Shape:
  def __init__(self, color):
  self.color = color
		
class Triangle(Shape):
  def __init__(self, color, base, height):
  super().__init__(color)
  self.base = base
  self.height = height
```
- Nếu lớp con không có thêm thuộc tính riêng nào thì có thể không cần khởi tạo nữa mà tự động kế thừa luôn phương thức khởi tạo của lớp cha
